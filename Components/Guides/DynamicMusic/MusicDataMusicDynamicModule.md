# Music Data : MusicDynamicModule

In MusicData (JSon) :

**“dynamicModules**” : is a key map of MusicDynamicModule. 

{
**"$type": "ThunderRoad.MusicDynamicModuleMap, ThunderRoad",**
 "Key" : {Put MusicDynamicModule Data here}
}
